How I Develop Software

I build a platform integrating Cloud Foundry, Kubernetes, and Istio. I specifically work in networking. I work primarily in golang and yaml. I write more tests than code, and the results of my labor are usually only visible in the terminal. I am impatient and lazy.

Philosophy

Goals:
  • fast as fuck
  • disposable
  • reproduceable
  • duplicatable
  • globally accessible
  • device agnostic
  • shareable with my teammates
  • functional for pair programming
Non-goals:
  • accessible offline
  • compatible with GUI applications

Hardware

  • e2-standard-8 VM on GCP (8 cores, 32gb RAM) -- the workstation
  • any computer (or, ipad, or phone) -- the client

Tools

  • SSH on the client to reach the workstation
  • Luan's Neovim Distribution for editing
  • tmux for window managing
  • chrome for web browsing on the client
  • concourse for rolling/duplicating the workstation

Repo

I built devenv, a reproduceable set of scripts that builds a fully working development environment in GCP from zero in under 15 minutes, and a concourse pipeline to redeploy environments on demand.
